Project Overview

This project was created in 2025 as part of the Junior Developer test task for Chili Labs.
Its purpose is to demonstrate the ability to work with a public API, structure a small application, and deliver a clear, functional implementation.

Tech Stack

The project uses the following technologies:

HTML

CSS / TailwindCSS

JavaScript / TypeScript

React.js

Next.js

Git

Features

Fetching data from a public API

Rendering a product list interface

Structured project layout

Clean, maintainable code

Simple, responsive UI design
